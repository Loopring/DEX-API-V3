Get L2 block info

流量控制: 5 每 秒

摘要

HTTP 方法

GET

访问路径

/api/v3/block/getBlock

功能

Get L2 block info by block id

API 描述

HTTP头

字段 类型 必须 说明 举例
X-API-KEY string 是 ApiKey "HlkcGxbqBeaF76j4rvPaOasyfPwnkQ
6B6DQ6THZWbvrAGxzEdulXQvOKLrRW
ZLnN"

请求参数

字段 类型 必须 说明 举例
id string 否 The block id, could be finalized, confirmed, or block_idx_num
默认值 : finalized
取值范围 : ['finalized', 'confirmed', '{block_idx_num}']		 "1234"

请求示例

HTTP
CURL
GET https://api3.loopring.io/api/v3/block/getBlock?id=1234 HTTP/1.1
Host: api3.loopring.io
Connection: keep-alive
Accept: */*
Accept-Encoding: gzip, deflate, br
Accept-Language: zh,en;q=0.9
X-API-KEY: sra1aavfa
curl -H "X-API-KEY:sra1aavfa" https://api3.loopring.io/api/v3/block/getBlock\?id\=1234
